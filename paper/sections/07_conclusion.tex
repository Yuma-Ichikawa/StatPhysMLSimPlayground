\section{Conclusion}

We have specified one falsifiable and reproducible numerical thermodynamics for modern
learning systems. The unifying object is not a metaphorical free energy but a continuation
path from an exact or oracle model, together with stable order parameters, matched disorder,
finite-size diagnostics, and registered interventions. The implementation covers the
Transformer M0--M8 and D0--D5 ladders, programs A--N, diffusion, reinforcement learning,
collective reasoning, and four cross-domain couplings. It enforces exactly five outer
seeds and prevents incomplete numerical values from entering the paper. The resulting
experiment is designed to answer both positive and negative questions: which phase
structures survive realism, which assumptions break them, and which apparent transitions
are only finite-size crossovers.
