\section{Reproducibility contract}
\label{app:reproducibility}

Each sweep starts from a TOML file and expands to a JSONL manifest whose rows
are complete \code{RunSpec} objects.  The canonical JSON hash determines a
16-character run identifier.  Slurm array index is merely a dispatcher and is
not used as a scientific seed.  Site-specific container locations are supplied
through an environment variable; source code and experiment definitions
contain no user-home path.

The artifact tree for one run contains \code{spec.json},
\code{provenance.json}, \code{status.json}, \code{summary.json}, compressed
trajectories and diagnostics, and the quenched teacher state.  Atomic rename
prevents partially written JSON/NPZ files from appearing complete.  Manifest
events are append-only under a file lock.  Aggregation verifies checksums and
records missing, failed, preempted, and non-finite runs.

\section{Exact fidelity tests}

The bridge test suite checks: (i) positional rows are exactly
$\pm\bm 1/\sqrt d$; (ii) teacher semantic logits and target values use raw
tokens; (iii) $\omega=0$ and $1$ recover pure semantic and positional targets;
(iv) all RNG streams leave global NumPy/PyTorch state unchanged; (v) semantic
and positional informed factors equal their intended directions; and (vi) the
summed loss matches a direct implementation.  Architecture tests additionally
check diagnostic shapes, Q/K tieing, cumulative feature gates, RoPE parity,
causal invariance, and $M8$ shifted pairs.
